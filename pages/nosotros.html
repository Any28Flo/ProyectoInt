El equipo de In Code ha sido el encargado del desarrollo de la aplicación web de tipo e-commerce para la empresa Never Dies Co. El objetivo del desarrollo de esta aplicación es poder implementar el sistema de ventas de todo tipo de diseños impresos en distintas prendas y obtener una mayor difusión para la marca.
A continuación se listan algunas de las características con las que cuenta el sistema:

  • CRUD para diseños
  • CRUD para usuarios
  • CRUD para carousel de novedades
  • CRUD para carrito de compras
  • CRUD para carrito de compras
  • CRUD para publicación de noticias o eventos
  • Panel de control de cliente
  • Panel de control de Administrador
  • Formulario de inicio de sesión
  • Formulario para crear cuenta nueva
  • Formulario de contacto
